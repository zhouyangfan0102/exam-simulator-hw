# 题库模拟考试

这是一个可在 Windows 或 macOS 本地运行的题库练习页面，不需要安装第三方 Python 包。

## 运行条件

- Windows 10/11 或 macOS 电脑。
- Python 3.10 或以上版本。
- 不需要联网，不需要安装 pip 依赖。

## 打包发送

把整个 `考试模拟器` 文件夹压缩发送给对方即可，至少要包含这些内容：

- `server.py`
- `Windows-启动考试模拟器.bat`
- `Windows-关闭考试模拟器.bat`
- `macOS-启动考试模拟器.command`
- `macOS-关闭考试模拟器.command`
- `templates` 文件夹
- `static` 文件夹
- `data` 文件夹

Windows 解压后双击 `Windows-启动考试模拟器.bat`；macOS 解压后双击 `macOS-启动考试模拟器.command`。按窗口里显示的地址打开浏览器，通常是 `http://127.0.0.1:8000`。使用结束后双击对应系统的“关闭考试模拟器”文件。

如果 macOS 首次运行提示文件没有执行权限，请在“终端”中进入项目目录后执行：

```bash
chmod +x macOS-启动考试模拟器.command macOS-关闭考试模拟器.command
```

项目必须先完整解压到桌面、文档等有写入权限的位置，不要直接在压缩包内运行，也不要放在 `C:\Program Files` 等受保护目录。当前版本只允许本机访问，不支持手机通过局域网打开。

如果要随包发送固定题库，请把最终 Excel 文件放在 `data` 文件夹中。也可以让对方打开页面后点击“上传题库”选择自己的 `.xlsx` 文件。

页面中的“整理题库”可以上传 `.docx` 或 `.xlsx` 原始题目清单，系统会自动下载一份整理后的标准 Excel。建议先检查整理结果，再用“上传题库”导入。

打包前建议关闭运行窗口，保存并关闭 Excel/Word。`templates`、`static`、`data` 都需要保留；`static\题库模板.xlsx` 是页面“题库模板”按钮使用的文件。`data\bank_meta.json` 会记录默认读取哪个题库，如果希望默认读取 `data\题库.xlsx`，请确认它指向 `题库.xlsx`，或删除该 JSON 后再打包。

## 使用方式

1. 把你的 Excel 题库文件命名为 `题库.xlsx`，放到 `data` 文件夹里。
2. Windows 双击 `Windows-启动考试模拟器.bat`；macOS 双击 `macOS-启动考试模拟器.command`。
3. 浏览器打开窗口里显示的地址，通常是 `http://127.0.0.1:8000`。
4. 勾选学科，选择随机出题或全量考试；开始后可点击“上一题”“下一题”、题号，或使用方向键切题，提交后会用绿色/红色标出正确和错误。
5. 使用结束后双击对应系统的 `Windows-关闭考试模拟器.bat` 或 `macOS-关闭考试模拟器.command`。

## Excel 格式

- 每个 sheet 页标签表示一门学科。
- 每个 sheet 的 A/B/C 列分别是：题型、答案、问题。
- 第一行可以写表头：`题型`、`答案`、`问题`，程序会自动跳过。
- 题型建议写：`判断题`、`单选题`、`多选题`。
- 判断题答案可以写：`对` / `错`、`正确` / `错误`、`TRUE` / `FALSE`。
- 单选题答案写选项字母，例如 `A`；多选题答案写多个字母，例如 `AC`。
- 选择题建议按模板分行填写，题型和答案写在题目那一行，选项另起一行只填 C 列，例如：

```text
A列：单选题 | B列：A | C列：下列属于偶数的是：
A列：      | B列：  | C列：A. 2
A列：      | B列：  | C列：B. 3
A列：      | B列：  | C列：C. 8
A列：      | B列：  | C列：D. 9
```

## 更新题库

后续只需要编辑并保存 `data\题库.xlsx`。回到网页点击顶部的圆形刷新按钮，或者重新开始考试，就会读取最新内容。

如果 Excel 正在被占用导致读取失败，先保存并关闭 Excel 后再刷新题库。
