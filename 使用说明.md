# 考试模拟器使用说明

## 一、软件说明

考试模拟器是一个在 Windows 电脑本地运行的题库练习工具，支持：

- 判断题、单选题和多选题。
- 多学科组合、随机 30 题或全量出题。
- 多个题库上传、切换和题目模糊搜索。
- 单题判定、题目标记、考试计时和交卷统计。
- 将常见 `.docx` 或 `.xlsx` 题目清单整理为标准题库。

程序只在当前电脑上运行，不需要联网，不会把题库上传到互联网。当前服务仅监听 `127.0.0.1`，其他电脑或手机不能通过局域网访问。

## 二、运行条件

- Windows 10 或 Windows 11。
- Python 3.10 或以上版本。
- 不需要安装第三方 Python 包，不需要执行 `pip install`。

可以在命令提示符中执行以下任意一条命令检查 Python：

```text
python --version
py -3 --version
```

如果没有显示 Python 3.10 或更高版本，请从以下地址安装：

```text
https://www.python.org/downloads/
```

安装时建议勾选 `Add python.exe to PATH`。

## 三、项目文件说明

正式运行至少需要保留以下内容：

```text
考试模拟器
├─ server.py
├─ 启动考试模拟器.bat
├─ 使用说明.md
├─ templates
│  └─ index.html
├─ static
│  ├─ app.js
│  ├─ styles.css
│  └─ 题库模板.xlsx
└─ data
   ├─ 题库.xlsx
   └─ bank_meta.json
```

文件作用：

- `server.py`：本地服务、Excel 读取、考试和搜索逻辑。
- `启动考试模拟器.bat`：Windows 启动脚本。
- `templates`：页面结构，不能删除。
- `static`：页面脚本、样式和下载模板，不能删除。
- `data`：存放所有题库 Excel 和当前题库记录，必须具有写入权限。
- `bank_meta.json`：记录下次启动时默认打开哪个题库；删除后会优先读取 `data/题库.xlsx`。

以下内容不是运行必需文件：

- `README.md`。
- `tools_create_sample_workbook.py` 等测试或生成脚本。
- `__pycache__` 文件夹和 `.pyc` 文件。
- 名称中带“测试”的临时题库。

## 四、首次启动

1. 先将压缩包完整解压，不要直接在压缩包内运行。
2. 建议解压到桌面、文档或其他有写入权限的位置，不要放在 `C:\Program Files` 等受保护目录。
3. 双击 `启动考试模拟器.bat`。
4. 等待命令窗口显示访问地址。
5. 在浏览器中打开该地址，通常为：

```text
http://127.0.0.1:8000
```

启动后不要关闭命令窗口，关闭窗口会停止服务。

如果 `8000` 端口已被占用，程序会自动尝试后续端口。请以命令窗口实际显示的地址为准，不要固定使用 `8000`。

## 五、题库 Excel 格式

题库必须是 `.xlsx` 文件。每个 sheet 页表示一门学科，sheet 页名称就是科目名称。

每个 sheet 页中：

- A 列：题型。
- B 列：答案。
- C 列：问题和选项。

第一行可以写表头：

```text
题型 | 答案 | 问题
```

支持的题型：

- 判断题。
- 单选题。
- 多选题。

答案示例：

- 判断题：`对`、`错`、`正确`、`错误`、`TRUE`、`FALSE`。
- 单选题：`A`。
- 多选题：`AC`，答案字母顺序不影响判定。

选择题建议按以下方式分行填写：

```text
A列：单选题 | B列：A | C列：下列属于偶数的是：
A列：      | B列：  | C列：A. 2
A列：      | B列：  | C列：B. 3
A列：      | B列：  | C列：C. 8
A列：      | B列：  | C列：D. 9
```

题型、答案和题干写在同一行，每个选项另起一行并只填写 C 列。

## 六、上传、整理和切换题库

### 下载题库模板

点击页面顶部“题库模板”，可以下载标准 Excel 示例。

### 整理原始题目

1. 点击“整理题库”。
2. 选择 `.docx` 或 `.xlsx` 文件。
3. 系统会下载一份名称带“整理后”的 Excel。
4. 打开文件检查题型、答案和选项。
5. 检查无误后点击“上传题库”导入。

整理功能不支持旧版 `.doc` 文件。自动识别适用于常见题号、A/B/C/D 选项和“答案：A”等格式，复杂排版仍需要人工检查。

### 上传和切换

- 点击“上传题库”可以导入 `.xlsx` 文件。
- 上传成功后，文件会保存在 `data` 文件夹中并自动切换为当前题库。
- 上传同名文件会更新原题库，重要题库建议提前备份。
- 页面顶部“当前题库”菜单会列出 `data` 中可读取的题库。
- 切换题库会清除正在进行的考试，请在开始考试前完成切换。

### 搜索题目

在“题目搜索”中输入题干、答案、科目或题型关键词。多个关键词可以用空格分隔，系统会在当前题库中进行模糊匹配。

## 七、考试规则和操作

默认随机题数为 `30`，系统会优先均衡抽取：

- 判断题：10 题。
- 单选题：10 题。
- 多选题：10 题。

如果所选题目不足 30 道，则按实际数量出题；某一种题型不足时，会使用其他可用题目补齐。

开始考试前可以：

- 选择一个或多个学科。
- 选择随机或全量模式。
- 设置全量题目是否乱序。
- 搜索并检查题库内容。

开始考试后：

- 顶部显示总题数、已答、正确、正确率和用时。
- 点击右侧题号可以切换题目。
- 点击“标记题目”可以添加或取消黄色标记。
- 点击“提交答案”会立即显示正确或错误。
- 点击“交卷”会统计整场考试。
- 点击右上角“重新开始”会返回初始页面。

当前考试记录保存在内存中。刷新页面、切换题库、关闭服务或重新启动电脑都会清除本次答题进度。

## 八、打包发送到其他电脑

直接压缩整个 `考试模拟器` 文件夹即可，不要只发送 `index.html` 或单独发送启动脚本。

### 打包前检查清单

1. 保存并关闭正在编辑的 Excel、Word 和代码文件。
2. 在页面中切换到希望对方首次看到的正式题库。
3. 确认 `data/bank_meta.json` 指向该正式题库；也可以删除该文件，让程序默认读取 `data/题库.xlsx`。
4. 删除不准备发送的测试题库、旧题库和重复题库。
5. 可以删除 `tools_create_sample_workbook.py`、`__pycache__` 和 `.pyc` 文件。
6. 删除 Excel 产生的 `~$` 开头临时文件。
7. 确认 `templates`、`static`、`data` 和 `static/题库模板.xlsx` 均存在。
8. 关闭考试模拟器命令窗口后，再压缩整个文件夹。

当前项目中的 `data/multi_subject_test.xlsx` 和 `tools_create_sample_workbook.py` 仅用于测试，不影响程序运行，也可以在正式打包前移除。

### 接收方操作

1. 将压缩包完整解压到桌面或文档目录。
2. 安装 Python 3.10 或以上版本。
3. 双击 `启动考试模拟器.bat`。
4. 按命令窗口显示的地址打开浏览器。

目前项目不是独立 `.exe`，目标电脑没有 Python 时无法直接运行。

## 九、备份和更新题库

- 日常备份时，复制整个 `data` 文件夹即可保存所有题库和当前题库设置。
- 直接编辑并保存 `data` 中的 Excel 后，回到页面点击顶部圆形刷新按钮即可重新读取。
- 上传同名题库前，建议保留原文件副本。
- 不要在程序读取或上传过程中移动、重命名或删除题库文件。

## 十、常见问题

### 1. 双击后提示没有 Python

安装 Python 3.10 或以上版本，并确保安装时勾选 `Add python.exe to PATH`。安装后重新打开项目文件夹再启动。

### 2. 页面打不开

- 确认命令窗口仍然打开。
- 使用命令窗口显示的实际地址。
- 如果 `8000` 被占用，程序可能运行在 `8001`、`8002` 等后续端口。
- 不要把 `127.0.0.1` 改成其他电脑的地址。

### 3. 题库读取失败

- 确认文件扩展名是 `.xlsx`，不是 `.xls`。
- 确认每个 sheet 的 A/B/C 列分别是题型、答案和问题。
- 保存并关闭正在编辑的 Excel 后再刷新。
- 使用“题库模板”对照格式。

### 4. 上传失败或无法保存当前题库

- 确认项目已经完整解压。
- 确认项目目录和 `data` 文件夹具有写入权限。
- 不要放在 `C:\Program Files`、只读磁盘或压缩包内部运行。
- 检查磁盘剩余空间，避免上传体积异常大的文件。

### 5. 搜索或题库菜单没有更新

先点击顶部圆形刷新按钮；仍未恢复时，关闭命令窗口并重新启动考试模拟器，然后刷新浏览器页面。

### 6. 手机无法打开电脑上的地址

这是当前版本的正常行为。服务只允许当前电脑访问，暂未开放局域网访问。

### 7. 中文路径是否支持

支持。项目可以放在带中文和空格的目录中，但目录必须具有读写权限。
